Installation Guide

Required Dependencies

For the TGS to WebP converter to work, you need to install the following Python packages:

pip install lottie pillow

Or for better quality and additional features:

pip install lottie[all]

Alternative: Using lottie-converter CLI

You can also install the lottie-converter command-line tool:

pip install lottie-converter

Running the Web Application

Once you have the dependencies installed, you can run the Flask application:

python app.py

Then open your browser and navigate to http://localhost:5000

Troubleshooting

If conversion fails, make sure:

  • All dependencies are properly installed
  • You have sufficient disk space
  • The input file is a valid TGS file
